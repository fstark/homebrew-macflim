# Homebrew Tap for MacFlim

This is the Homebrew tap for [macflim](https://github.com/fstark/macflim) - a tool to create classic Macintosh FLIM movies from modern video files.

## Installation

```bash
brew tap fstark/macflim
brew install flimmaker
```

Or in one command:

```bash
brew install fstark/macflim/flimmaker
```

## Usage

```bash
flimmaker input.mp4 [OPTIONS]
```

For full documentation, see the [main repository](https://github.com/fstark/macflim).

## Updating

```bash
brew update
brew upgrade flimmaker
```

## Uninstalling

```bash
brew uninstall flimmaker
brew untap fstark/macflim
```
